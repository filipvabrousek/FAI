## Static elements
* no need to create instances

```cs

 // STATICKÉ PRVKY
    static class Counter {
        public static int count(int x, int y) {
            return x + y;
        }
    }
    
    
    
     // STATICKÉ PRVKY
     int sum = Counter.count(6, 3); // bez nutnosti vytváření instance
     print(sum + "");






```
